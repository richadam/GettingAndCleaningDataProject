# Code book for Getting and Cleaning Data course project

The resultant data set is located in the `tidy_data.txt` file in this repository.

The `README.md` file in this repository contains additional information on this project.

The data set is described in the [Data](#data) section, the variables are listed in the [Variables](#variables) section, and the steps required to create the `tidy_data.txt` file from the source data are presented in the [Instructions](#instructions) section.

## Data <a name="data"></a>

The `tidy_data.txt` data file is a text file, containing space delimited values.

The first row contains the names of the variables, which are listed and described in the [Variables](#variables) section, and the remaining rows contain the values for each of these variables. 

## Variables <a name="variables"></a>

Each row contains 79 averaged signal measurements for a given subject and activity.

### Identifiers <a name="identifiers"></a>

- `subject`

	Subject identifier, integer, ranges from 1 to 30.

- `activity`

	Activity identifier, string with 6 possible values: 
	- `WALKING`
	- `WALKING_UPSTAIRS`
	- `WALKING_DOWNSTAIRS`
	- `SITTING`
	- `STANDING`
	- `LAYING`

### Average of measurements <a name="average-measurements"></a>

All measurements are floating-point values, normalized and bounded within [-1,1].

Prior to normalization, acceleration measurements (variables containing `Accelerometer`) were made in *g*'s (9.81 m.s⁻²) and gyroscope measurements (variables containing `Gyroscope`) were made in radians per second (rad.s⁻¹).

Magnitudes of three-dimensional signals (variables containing `Magnitude`) were calculated using the Euclidean norm.

The measurements are classified in two domains:

- Time-domain signals (variables prefixed by `timeDomain`), resulting from the capture of accelerometer and gyroscope raw signals.

- Frequency-domain signals (variables prefixed by `frequencyDomain`), resulting from the application of a Fast Fourier Transform (FFT) to some of the time-domain signals.

## Instructions <a name="instructions"></a>

The zip file containing the source data is located at [https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip](https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip).

The following steps will create the `tidy_data.txt` file in the current working directory:
1. source("run_analysis.R")

This will:
1. Download and read in the data
1. Merge the training and test sets into one data set.
1. Extract only the measurements on the mean and standard deviation for each measurement.
1. Replace the activity identifiers with descriptive activity names.
1. Replace the variable names with descriptive variable names.
1. Create the tidy_data.txt with the average of each variable for each activity and each subject.
